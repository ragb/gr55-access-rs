# gr55-core

Pure no-IO codec for Roland GR-55 SysEx messages and patch data.

Mirrors the methodology of the sibling `re202-core` crate. Adapted to the GR-55's
larger address space and the address map maintained by Colin Willcocks' GR-55
FloorBoard project (vendored under `data/midi.xml`).

Under construction. See `../docs/sysex-notes.md` for the current state of
protocol knowledge and `../docs/gr55-bootstrap.md` for project intent.
